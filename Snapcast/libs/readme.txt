If you want to use local AAR packages, and not download them from GithubPackages, 
place them in this directory:

- boost-1.85.0
- flac-1.4.2
- oboe-1.9.0
- ogg-1.3.5
- opus-1.1.2
- soxr-0.1.3
- tremor-1.0.1
- vorbis-1.3.7

you can download the aar files from here:
https://github.com/badaix/snapcast-deps/releases/tag/v0.29.0

If you want to use the GithubPackages repository, you must provide a github user and token 
with read:packages permission via the environment variables GITHUB_USER and GITHUB_TOKEN
